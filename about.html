Over deze website

Deze website is gemaakt om zo veel mogelijk mensen te informeren over Machine Learning, we willen dat meer mensen over Machine Learning te weten komen omdat technologie namelijk steeds belangrijker wordt en Machine Learning is een van de dingen die steeds beter ontwikkelt wordt en later heel belangrijk wordt.


Door wie is deze website gemaakt?

Deze website is gemaakt door Mikail Sola en Walat Hamasahid, wij zijn allebei leerlingen van het Sint-Maartenscollege in Voorburg en we zitten allebei in 4 Havo. Deze website is gemaakt voor een PO waar we zelf mochten kiezen over welke onderwerp we de website zulde hebben met het doel om onze vaardigheden van coderen te verbeteren, om onze fouten te herkennen tijdens het coderen en zelf namelijk opzoeken wat de fouten zijn en het dan ook zelf verbeteren.

We hebben tijdens allerlei dingen geleerd:

  • Hoe je moet werken met HTML
  • Hoe je moet werken en stylen met CSS
  • Hoe je jezelf een planning kan geven met behulp van Trello
  • Hoe je moet werken met Replit
  • En hoe je je progressie van je website kan bekijken met behulp van GitHub
