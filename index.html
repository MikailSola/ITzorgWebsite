Wat is Machine Learning?

Machine learning (ML) is de studie van computer algoritmen die automatisch kunnen verbeteren door ervaring en door het gebruik van gegevens. Machine Learning is een vorm van Artificial Intelligence (AI).Tegenwoordig wordt machine learning overal om ons heen toegepast. Bij de communicatie met banken, het online winkelen of het gebruik van sociale media draagt machine learning ertoe bij dat onze ervaringen efficiënt, soepel en veilig verlopen. Machine learning en de technologie eromheen ontwikkelen zich snel, maar we beginnen pas net te begrijpen welke mogelijkheden ze allemaal te bieden hebben.
